## 1. Merge란?

**Merge**

- a 브랜치에서 작업했던 내용이 m에서 유용할 것 같을 때 **Merge**

  a 브랜치와 m 브랜치에서 작업한 내용을 동시에 갖는 새로운 버전을 만들고 싶을 때 사용

  합치려고 하는 공통 조상 = base (3번째 point)

  이렇게 합쳐서 만든 버전을 merge commit이라고 부름
